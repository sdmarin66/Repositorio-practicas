# Sergio - GitHub SMR 🐍

Repositorio personal creado para las prácticas del ciclo formativo de **Sistemas Microinformáticos y Redes (SMR)**.

El objetivo de este proyecto es aprender a trabajar con **Git y GitHub**, utilizando ramas independientes para organizar la información y gestionar entornos de trabajo.

---

## 🚀 Objetivos del proyecto

* **Aprender a utilizar GitHub** como plataforma de trabajo profesional.
* **Practicar la creación de ramas** (`prod` y `sergio`) para aislar directorios.
* **Gestión de terminal:** Dominar comandos de Linux para la administración de archivos.
